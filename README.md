# 🐍 Snake Game (Single HTML)

HTML 파일 **하나**로 실행되는 간단한 스네이크(Snake) 게임입니다.  
**CSS + JavaScript**를 모두 한 파일에 포함했으며, **키보드 화살표 키(↑ ↓ ← →)** 로 조작합니다.

> ✅ 실행: `snake.html`을 브라우저로 열면 바로 플레이할 수 있습니다.
